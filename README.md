## üèóÔ∏è Architecture & Event Flow

Instead of storing current state in a CRUD-style model, the system
writes **events**:

    HouseRegistered
    WifiSignalCaptured
    RoomPerformanceCalculated
    PerformanceRecommendationGenerated

Read models are generated by replaying these events.
